prepare and send a PDU for this session

Parameters
paththe path of the resource
payloadthe payload to send with the PDU
pdu_typerequest confirmation of the request (CON) or not (NON)
codethe code similar to HTTP (e.g., GET, POST, PUT, ...)
observeobserve/subscribe the resource
querysend a query - comparable to path?arg1=val1&arg2=val2 in HTTP
optionsset additional options (e.g., COAP_OPTION_CONTENT_FORMAT) using a list of (option_code, value) tuples
save_rx_pduautomatically make the response PDU persistent
response_callbackfunction that will be called if a response is received
response_callback_dataadditional data that will be passed to response_callback
Returns
the resulting dictionary in token_handler
